# 文件操作(file.txt)

* 查阅文件行数： wc -l file.txt
* 查找关键词: grep <keyword> file.txt (keyword不能多个单词)
* source


# 内存磁盘管理

* 检查内存使用情况：free -h
* 检查磁盘使用情况：df -sh
* 检查某个目录下的磁盘使用情况： du -sh <directory>
* top/htop

# 进程管理

* ps aux | grep <name>
* kill -9 <pid>

# 权限管理

* groupadd/useradd/newgrp/chown/chmod

# 其他

* export
